# API to process tasks

## Description
 - An api to schedule & process tasks.

## Setup
- Copy contents for .sample_env file and replace with correct values
- And run the following commands from your terminal
```bash
git clone <url>
cd task-schedule-node
npm install
npm start
```
## Enpoints
